Project 1: Unionized & Nonunionized Worker Wage Data Analysis Project

Description of Project 1. This project was built using python and python based libraries. It observes wage differences between unionized and non-unionized workers in the state of Michigan in the manufacturing industry, and aims to see if there is a correlation between union membership in the manufacturing industry in the state of Michigan and wage differences. This project utlizes visualizations shown below.

Libraries Used

  • Pandas
  • Matplotlib
  • Numpy
  • Scipy
  • sklearn

    • Statistical Methods Used

    • bivariate t-test
    • Linear Regression

Challenges and Solutions

During the development of this project, I faced several challenges such as cleaning the data, understanding the dataset documentation, and drawing the appropriate conclusions based off of my statistical results. To overcome these challenges, I made extra efforts to understand the data, re-labeled the variables for better readabilty, and preformed extensive ressearch on Matplotlib documentation so I could make the appropriate graphs.
