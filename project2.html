Project 2: Quantitative Methods in the Social Sciences Rurality in Michigan Project

This project illustrates how rurality is distributed across the state of Michigan, and how wage is impacted.

Technologies Used

  • Tableau
  • Microsoft Excel

Challenges and Solutions

During the development of this project, I faced several challenges such as data cleaning, understanding the documentation, and understanding how to properly display infromation in a professional manner. To overcome these challenges, I implemented solutions such as reading the documentation of the data , classifying the excel data accordingly in order for Tableau to read it correctly, and use specific color pallates in my visualizations to further enhance my data.
